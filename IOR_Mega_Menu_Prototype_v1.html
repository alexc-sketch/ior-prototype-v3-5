IOR Mega Menu Prototype v1

This file is a standalone functioning reference for the IOR navigation system. Click any nav item above to open the full mega panel. Resize below 1200px to see the mobile drawer.

The Digital Platforms panel is the add-ons menu — HyDip™ FMS, Fuelcharge, and Customer Portal are positioned as product add-ons to the core IOR fuel account. Each column has a product descriptor, key links, and a direct login/download CTA.

Homepage Audit — Issues Found
Issue Severity Detail
Hero uses linear-gradient overlay FAIL R3 Line 1378: style="background-image:linear-gradient(rgba(3,85,163,.70)...)" — Golden Rule R3: no gradients. Hero should use a solid navy background or a real video/photo.
150 inline style= attributes FAIL R4 Golden Rule R4: no inline style= on CTAs or layout elements. Most are on annotation blocks (acceptable) but several are on live UI elements including section backgrounds, mega panel columns, and CTA buttons.
Footer hrefs use v3.5 filenames FAIL Lines 1870, 1908–1909: 11b_Regional_Contacts.html, 11e_Make_Payment.html, 11a_Contact_Us.html, 11_Support_Hub.html — should be v5 equivalents.
Footer credit: Site by 3PM FAIL Line 1931: should read Site by ThreePM.
Mega menu panels are annotation blocks, not live nav STRUCTURAL The homepage mega menus (Blocks 1.3–1.7) are rendered as static annotation previews inside <div style="background:#f7f9fc..."> wrappers — they are not wired to the actual nav buttons. The live nav (lines 929–934) has 5 buttons but the mega panels are not connected to them.
Nav has duplicate scrolled-state demo block STRUCTURAL Lines 956–990: a second static nav demo (scrolled state) is rendered as a visual annotation inside the page body. This is intentional as a design reference but should be noted for Elementor build.
Quick Access Bar uses emoji icons + inline styles MINOR Lines 1429–1460: qa-icon divs use style="font-size:28px;background:..." inline. Should use SVG icons with CSS classes.
Fuelcharge teal cell in Quick Access Bar REVIEW The teal qa-cell--teal uses Fuelcharge brand colour on a global component. Per branding rules, bespoke brand colours should not appear on global nav/footer elements. Confirm if this is intentional.
26 border-radius occurrences MINOR Golden Rule R2: no rounded corners. Most are on .btn--pill (allowed) but some may be on card/image elements — needs line-by-line review.
Footer 11_Support_Hub.html link in mega panel FAIL Line 1212: Support mega panel links to v3.5 11_Support_Hub.html — should be 11_Support_Hubv5.html.
