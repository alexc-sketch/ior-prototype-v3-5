## 3. Component Flow & Structure

The Elementor Single Post Template must match this top-to-bottom visual flow:

### Section 1: "Back to Leadership" Utility Link
*   **Style:** A clean, subtle text link or button sitting above the bio.
*   **Copy:** `← Back to Our Leadership`
*   **Link:** Anchors back to the main Our Leadership page (`09a_Our_Leadershipv5.html`).

### Section 2: The Bio Profile Hero (Split Section)
*   **Layout:** A clean 2-column layout (Image on one side, Text/Details on the other based on design reference).
*   **Column 1 (Image):** Dynamic tag -> `ACF: Headshot Image`.
*   **Column 2 (Details):**
    *   **H1 (Name):** Dynamic tag -> `Post Title`.
    *   **H2 / Sub-header (Title):** Dynamic tag -> `ACF: Job Title` (Color: `var(--ior-blue)`).
    *   **Social Link:** LinkedIn Icon linked to `ACF: LinkedIn Profile` (set to hide if empty).
    *   **Bio Body:** Dynamic tag -> `ACF: Biography Content`.

### Section 3: Trust Flow (Related Content — Tabbed Grid)
*   **🚨 CRITICAL UPDATE:** The v3.5 prototype uses an outdated and broken blog grid container. You MUST delete the old version.
*   **Replacement:** Inject the correct Related Content — Tabbed Grid from `00_Global_Components.html#related-content`.
*   **Style:** Max 3 cards per tab (Case Studies & Blog Posts).
*   **Eyebrow:** `LATEST FROM IOR`

### Section 4: Pre-Footer
*   **Style:** Inherited strictly from the `IOR_Global_Template_Shell_v4.html` ("We're here to help").

### Section 5: Global Footer
*   **Style:** Append the standard v4.0 6-column global footer to close out the page.
