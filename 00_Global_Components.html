Primary Nav

Global Shell

White 92px nav bar. IOR logo left. Five mega-menu items centre. Two CTA buttons right. Do not add, remove, or reorder nav items. Do not modify the mega-menu HTML structure.

Nav CTA specification (Figma-confirmed):
  • Customer Login — .btn .btn--outline .btn--pill — border: 1px solid #0355A3, text #0355A3, background transparent, padding 22px 28px, border-radius: 500px, font-weight: 800
  • Apply for an Account — .btn .btn--primary .btn--pill — background #0355A3, text white, padding 22px 28px, border-radius: 500px, font-weight: 800
  • Font: Avenir Next LT Pro (licensed). Web fallback stack: 'Avenir Next', 'Nunito', 'Inter', sans-serif
  • Arrow icon: 10×10px rotated square outline (chevron-right equivalent) in matching colour
Five nav items (order locked): Fuel Solutions | Digital Platforms | Industries We Serve | About IOR | Support & Contact. Each triggers a mega-menu panel on hover/focus. Do not rename or reorder.
Elementor note: In WordPress, implement using Elementor Pro Nav Menu widget with custom mega-menu panels via Elementor Popup or JetMenu. The prototype nav is a static HTML reference only.

Mobile Drawer

Global Shell

Off-canvas slide-in navigation for viewports below 1024px. Triggered by the hamburger button in the nav bar. Contains all five nav items as accordion-style expandable sections, plus the two CTA buttons.

Do not modify the drawer HTML. The drawer is locked in IOR_Global_Template_Shell_v4.html. Copy the <div class="drawer"> block verbatim. The JS that opens/closes it is in the global script block at the bottom of the shell.
Elementor note: In WordPress, the mobile drawer is replaced by Elementor Pro's responsive nav menu with hamburger toggle. Ensure the two CTA buttons (Customer Login + Apply for an Account) are included in the mobile menu as full-width pill buttons.

Quick Links Bar

Global Shell

Contextual in-page link strip. Position: immediately after the hero </section>, before the stat bar. Not sticky — it scrolls away with the page. Only the primary nav is sticky. Set aria-current="page" and quick-links-bar__link--active on the current page link.

Ground Fuels quick links must always list all 5 spokes in order: IOR Diesel Network | On-Site Refuelling | Bulk Diesel Delivery | HyBlue™ | Oils & Lubricants. All links use _v3.5.html filenames.

Related Content — Tabbed Grid

Page Component

Case study and blog post grid with tabbed navigation. Exactly 3 cards per tab. Eyebrow must always read LATEST FROM IOR. The <h2> must be bespoke to the page audience.

Header rule: The <h2> MUST call out the specific target audience of the page. Example: "Case studies & insights for fleet operators." Do not use generic headers like "Read our blog."
Tab IDs: Use data-tab="rc-[UNIQUE_ID]-cases" and data-tab="rc-[UNIQUE_ID]-blog" where [UNIQUE_ID] is a short page slug (e.g. diesel, onsite). This prevents ID collisions when multiple components appear on one page.
